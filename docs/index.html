⚙️ 環境設定（首次使用必看）

🐍 Python 3.10+

大多數腳本需要 Python 3.10 以上。安裝完後執行：

pip3 install openpyxl rumps requests

☕ Java 8（EPB 必要）

EPB 資料橋接需要 JDK 1.8。確認路徑：

/Library/Java/JavaVirtualMachines/jdk1.8.0_251.jdk/

📡 EPB Shell

從公司 EPBrowser 取得 shell.jar，確認路徑：

/Library/EPBrowser/EPB/Shell/shell.jar

🔑 EPB 帳號

需要 EPB 系統帳號（公司內部）。Java 橋接使用 SOAP 直連，無需另外設定。

📂 腳本取得方式：向 Bob 索取腳本資料夾，或從本 repo 下載。
各腳本中的 /Users/bob.1469/ 路徑需改成你自己的使用者路徑。

KPI 月報
月報 每月手動
▶
計算北一區六門市員工月 KPI（原廠毛利、副廠毛利、SACare、ARpedia、喇叭、CPU）及夥伴/店長獎金， 輸出全版 + 門市版兩個互動式 HTML。

資料夾結構： 
KPI計算/
├── 執行.command          ← 雙擊執行
├── kpi_calculator.py
├── 輸入/
│   ├── 目標/             ← 放目標 Excel（月初設定一次）
│   └── 本月報表/         ← 放三份 EPB 報表
└── 結果/                 ← HTML 輸出
月初作業（一次）：
1
開啟 輸入/目標/北一區KPI目標設定模板.xlsx
2
填六門市原廠/副廠/SACare 毛利目標（未稅）
3
欄 L 填每位員工 CPU 月目標台數
月中/月底執行：
1
從 EPB 匯出 Report 13、Report 14、原始交易明細到 輸入/本月報表/
2
雙擊 執行.command
3
等 ~30 秒，瀏覽器自動開啟
⚠️ 毛利數字為月中追蹤用（不含教育價/回補），非公司正式薪資計算，月底以公司核定數字為準。

SACare 金額計算
月報 每月手動
▶
計算 SACare 搭售毛利金額。

公式：銷售金額 × 50% ÷ 1.05（未稅毛利）

使用步驟：
1
從 EPB 匯出 SACare 項目查詢 xlsx（個人或門市皆可）
2
放入 SACare計算/計算檔案/
3
雙擊 SACare計算/執行.command
輸出：SACare計算/結果/SACare金額結果_YYYYMMDD_HHMM.xlsx
計算完來源檔自動移至 已處理/

⚠️ EPB 直查規則：brand_id='999'、cat3_id='3012'、TRANS_TYPE IN ('A','E','H')
· 排除 G（訂金），否則分期商品重複計算
· 使用 北一區週報-app/data/SAcare對應價目表.xlsx（320 筆），勿用 644 筆版本

MyCoach 週報自動化
GitHub Actions 每週一 11:00 自動 ⚠️ 季度 2026-06-27 結束需更新
▶
每週一台灣時間 11:00 自動抓 MyCoach 各門市完課數，計算季度達成率，發送 Discord 通知。

技術架構：GitHub Actions Self-hosted Runner（本機 Mac）， 讀取本機 Chrome Cookie 繞過 MyCoach 驗證碼限制。

手動觸發： 
gh workflow run weekly_report.yml
Repo：Harveyzxc15/mycoach-report

⚠️ 季度結束需更新 mycoach_report.py：
SEASON_START、SEASON_END、SEASON_WEEKS
目前：SEASON_END = 2026-06-27
Session 過期時：在本機 Chrome 重新登入 MyCoach，下次 workflow 自動讀取新 Cookie。
Runner 重開機後需重新執行 ~/actions-runner/run.sh。
